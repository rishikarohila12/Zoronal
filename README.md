# Review & Rate App

A full-stack MERN application where users can explore companies, add companies, submit reviews and ratings, and interact with reviews through likes.

---

## 🚀 Features

- Add new companies
- Search companies
- Filter companies by city
- Sort companies
- Add reviews and ratings
- Dynamic average rating calculation
- Like functionality for reviews
- Responsive UI
- Toast notifications
- MongoDB integration
- REST API integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Deployment
- Frontend: Vercel
- Backend: Render
